# Reports

Rendered versions of the HTML reports in this directory are served via GitHub Pages:

| Report | Generated by | Description |
|---|---|---|
| [Backprop alternative on an easy inverse problem](https://yaroslavvb.github.io/inverse-problems/reports/linattention_report.html) | `altprop_linattention.py`, `linattention_visualize.py` | Alternative backpropagation with explicit layer-wise targets vs classic backprop on a 3-layer linear attention student/teacher |
| [Orthogonality-exploiting updates](https://yaroslavvb.github.io/inverse-problems/reports/ortho_updates_report.html) | `ortho_updates.py`, `ortho_initial_convergence.py` | 14 update rules exploiting the teacher's orthogonal weights (multiplicative expm/Cayley rotations, so(n)-Adam, retractions, landing, Procrustes) vs Euclidean SGD/Adam; determinant obstruction; initial-convergence re-tuning |
| [The two-block exact solver](https://yaroslavvb.github.io/inverse-problems/reports/two_block_report.html) | `two_block_report.py` | Detailed derivation and diagnostics of the alternating Procrustes / least-squares solver that recovers the teacher in ~10 alternations |
| [Learning without Forgetting](https://yaroslavvb.github.io/inverse-problems/reports/forgetting_report.html) | `forgetting_lab.py`, `forgetting_pretrained.py` | Muon's spectral descent read as worst-case forgetting control; ten anti-forgetting methods (incl. Tilde's compositional Muon) in a streaming student/teacher setting, plus a rerun on real pre-trained pythia-14m attention circuits |

The PNG/JSON files alongside the reports are the raw artifacts (plots and per-config
results) each report embeds and cites; the `.py` scripts in the repo root regenerate
them (`uv run <script>`).
